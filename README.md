# Amazon Q Developer in Action: Supercharging Application Development

## Overview

This repository contains materials and code examples from the presentation "Amazon Q Developer in Action: Supercharging Application Development" given at the AWS Kansas City User Group meetup. The presentation demonstrates how Amazon Q Developer can assist in various aspects of AWS development, from code generation to security enhancement and documentation.

For the presentation slides and additional information, [see](https://d1vse5fs7nznkq.cloudfront.net/).
